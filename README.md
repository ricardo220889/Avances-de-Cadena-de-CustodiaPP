<h2 align="center"> Descripción </h2>

Esta aplicación móvil está diseñada para policías y peritos, con el objetivo de registrar y documentar digitalmente la Cadena de Custodia desde un dispositivo Android.

El proyecto busca mejorar la organización, control y seguimiento de los indicios, sustituyendo procesos manuales por un sistema digital más eficiente y accesible.


<h2 align="center"> Objetivo </h2>

Desarrollar una aplicación móvil que permita el registro, control y seguimiento digital de la Cadena de Custodia, facilitando las tareas operativas del personal policial.

<h2 align="center"> Alcance </h2>

La aplicación permitirá:

-Registrar datos de la Cadena de Custodia

-Almacenar información de indicios

-Generar documentación digital

-Consultar registros creados

-Simular el proceso completo de Cadena de Custodia

<p><strong>Nota:</strong> No sustituye sistemas oficiales institucionales.</p>
